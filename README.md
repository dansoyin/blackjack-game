# Python Blackjack Game

A simple CLI Blackjack game built with Python.
Play against the computer using basic rules.

---

##  Features

```text
- Random card draw
- Automatic score calculation
- Ace can be 1 or 11
- Simple computer behavior
```

---

##  Run

```bash
python black_j.py
```

---

## 📌 Rules

```text
- Get as close to 21 as possible
- Blackjack (21 with 2 cards) is the best
- If you go over 21 → you lose
```
